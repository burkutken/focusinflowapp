# **App Name**: TaskZen

## Core Features:

- Task Creation: Allow users to create tasks with titles, descriptions, and importance levels.
- Task Ranking: Rank tasks based on importance level, deadline and show to the user.
- Task Display: Display current, next and other available tasks according to importance level and provide ability to start working on current/next task.
- Pomodoro Timer: Pomodoro timer that switches between work (25-60 mins) and break (5-25 mins) intervals. Settings stored via browser storage
- Fullscreen Timer: Show a fullscreen Pomodoro timer, which dims other apps for increased focus.
- Session Tracking: Tracks completed work/break sessions and saves those in localstorage to give daily, weekly, and monthly stats.
- Suggested timeframe: Use a tool to suggest an optimal timeframe between current time and task deadline to start working on the current and upcoming task, so the deadline meets.

## Style Guidelines:

- Primary color: Teal (#008080) for a sense of focus and calm.
- Background color: Light teal (#E0F8F8) for a gentle and focused ambiance.
- Accent color: Coral (#FF7F50) to highlight urgent tasks and important stats, to draw attention.
- Body and headline font: 'Roboto', a smooth, modern sans-serif font for readability.
- A clean, user-friendly layout to promote focus.
- Use intuitive icons for task categories and actions.
- Use subtle animations to show countdown timer and give gentle notification.
